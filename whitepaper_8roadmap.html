Geographical Scope

GasClick keeps track of the status of LPG consumption and crypto adoption in all countries to identify targets for deploying GasClick Platform and CRYPTOGAS. The map below represents the different regions that we have currently identified. Please, pay attention that this is subject to change at any time.

Addressable Market

All countries consume LPG and therefore all countries can be target to enjoy a LPG backed commodity currency as CRYPTOGAS. However, not all countries have the same level of freedom for cryptocurrencies. We consider addressable countries those that have a fair amount of LPG orders and are eager for cryptocurrencies.

Deploying a Country

Every country deployment is a project on its own and can require a different approach. We cannot define an unique country deployment framework, but we will define instead a set of principles and a suite of strategies.

Products

Entering a new country means to deploy the products below:

GasClick Platform

This is the core product to introduce in a new country. It includes technology, processes and know how.

Cards

We need to test some good open source card product and adapt to our needs to hold CYGAS payments. Also merchandising.

Discounts in shops

Agreement with merchants to provide discounts in shops paying with CYGAS card will help to value accrual for CYGAS..

Principles

Some principles to design specific country deployment plan are:

  • Delegation
  • Automation

Strategies

Some strategies are:

  • Work with existing efficient retailers
  • Enroll and train new retailers
  • Set a local retailer
  • Create a filling company

Performance Pattern

With these ideas in mind, we can create an initial reference success estimation to enable further planning for a template country. Obviously, data needs to be adjusted in a case by case basis.
